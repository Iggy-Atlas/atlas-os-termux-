Atlas is an advanced, personalized AI Operating System designed specifically for the **Termux** environment on Android. It bridges the gap between mobile terminal efficiency and high-level AI cognition.

---

### 🧠 Key Features (v17.0 Cognitive Update):
- **Vision Dual-Core**: Seamless image analysis using Groq (Llama 3.2) with an automatic Gemini 1.5 Flash failover.
- **Extended Memory**: Conversation history expanded to **20 messages** for deep context retention and complex reasoning.
- **Language Precision**: Optimized for standard literary languages, ensuring high-quality NLP across Croatian, English, and beyond.
- **Wiki-Intelligence**: Real-time Wikipedia integration for factual grounding.
- **PWA Integration**: Install Atlas as a standalone app on your Android home screen.

### 🛠️ Installation & Setup:
1. **Clone the repository:**
   ```bash
   git clone https://github.com/Iggy-Atlas/atlas-os-termux
   cd atlas-os-termux
   ```
2. **Install dependencies:**
   ```bash
   pip install -r requirements.txt
   ```
3. **Configure Environment:**
   Create a `.env` file with your `GROQ_API_KEY` and `GEMINI_API_KEY`.
4. **Launch Atlas:**
   ```bash
   python main.py
   ```
